#  MyGarage - Car Manager

![Java](https://img.shields.io/badge/Language-Java_17_LTS-orange) ![Framework](https://img.shields.io/badge/Framework-JavaFX-blue) ![Status](https://img.shields.io/badge/Status-Completed-success)

**MyGarage** is not just a standard inventory app; it's a specialized personal assistant for car enthusiasts. Built with **Java 17 LTS** and **JavaFX**, it offers a modern, innovative interface to manage your garage, track racing sessions, a detailed wishlist and analyze expenses visually.
